# KIKI Mods

BepInEx Mods for KIKI.

## Prerequisites
Install [BepInEx](https://github.com/BepInEx/BepInEx) first.
Then drag & drop the .dll mods on `/BepInEx/plugins`.
It's all done!

## Mods
- **CoyoteTime** — Adds a 150ms coyote time to the jump
- **PressRtoRestart** — Press R to restart the level
