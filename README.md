# KIKI Mods

BepInEx Mods for [KIKI](https://store.steampowered.com/app/2142680/Kiki/).

## Prerequisites
1. Install [BepInEx](https://github.com/BepInEx/BepInEx/releases/latest).
2. Download the ZIP files for the mods you want to install.
3. Unpack the ZIP at the game's root.
5. It's all done!

## Mods
- **CoyoteTime** — Adds a 150ms coyote time to the jump
- **PressRtoRestart** — Press R to restart the level
