# KIKI Mods

BepInEx Mods for KIKI.

## Prerequisites
1. Install [BepInEx](https://github.com/BepInEx/BepInEx).
2. Drag & Drop the .dll mods on `/BepInEx/plugins`.
3. It's all done!

## Mods
- **CoyoteTime** — Adds a 150ms coyote time to the jump
- **PressRtoRestart** — Press R to restart the level
